ClearStream

Detects the video stream on the page you're on and plays it in a clean, ad-free player — no popups, no fake buttons, no malvertising. Runs entirely in your browser: no servers, no analytics, no accounts.

Install in one command:

git clone https://github.com/leocgbraga/clearstream && cd clearstream
./install.sh
